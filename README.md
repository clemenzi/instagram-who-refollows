# Who refollows me?

A lightweight browser extension to find the Instagram profiles you follow that do not follow you back. Built with React, Tailwind CSS v4, and WXT.

## What it does

Open Instagram, start a scan, and get a clean list of accounts that are in your following list but not in your followers list.

- Reads followers and following from the Instagram tab you already have open
- Shows scan progress while it collects profiles
- Opens the result directly inside Instagram
- Keeps the logic in the browser extension, without a backend

## Privacy

The extension runs from your browser while you are logged into Instagram. It does not ship with a server and does not store your Instagram data remotely.
